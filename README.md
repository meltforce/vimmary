YouTube video summary service. Fetches transcripts via YouTube's InnerTube API, generates LLM summaries (Claude or Mistral), stores everything in Postgres + pgvector for semantic search.

See [codeberg.org/meltforce/vimmary](https://codeberg.org/meltforce/vimmary) for current code, releases, and issues. Live demo: https://vimmary.meltforce.org/
